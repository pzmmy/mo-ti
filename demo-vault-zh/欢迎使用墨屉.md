# 🎉 欢迎使用墨屉！

墨屉是你的**知识抽屉**——一个本地优先、Markdown 原生的知识库管理工具。

## 快速上手

### 1. 创建第一篇笔记
按 `Ctrl+N` 或点击左侧的「新建笔记」按钮。

### 2. 使用 Markdown 写作
```markdown
# 标题
## 二级标题

**粗体** *斜体* ~~删除线~~

- 列表项 1
- 列表项 2

1. 编号列表
2. 编号列表

> 引用一段话

[链接文字](https://example.com)
```

### 3. 连接笔记
在笔记中输入 `[[` 可以链接到其他笔记——这是构建知识图谱的基础。

### 4. 使用 AI
墨屉内置了 DeepSeek、通义千问、智谱 GLM 等国产 AI。在设置中添加 API Key 后，按 `Cmd+K` (Mac) 或 `Ctrl+K` (Win) 打开命令面板即可使用。

### 5. 同步笔记
- **Git 同步**：适合技术用户，版本管理更完善
- **WebDAV 同步**：适合普通用户，配合坚果云/Nextcloud 使用
- **手机伴侣**：浏览器打开 https://pzmmy.github.io/mo-ti/mobile/ 即可在手机上浏览编辑
