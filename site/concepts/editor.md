# 编辑器

Tolaria 提供适合日常写作的富文本编辑器，以及用于精确文件控制的原始 Markdown 模式。两种模式都写回同一个 Markdown 文件。

## 富文本编辑

富文本编辑器支持块、斜杠命令、维基链接、表格、代码块、图片、Mermaid 图表、LaTeX 风格数学公式和基于 Markdown 的白板。

当你想要快速写作和重新组织而不必考虑 Markdown 语法时，请使用此模式。

## 原始模式

原始模式直接显示 Markdown 源码。当你需要编辑 YAML frontmatter、修复不规范的 Markdown 或进行精确的文本修改时，请使用此模式。

在 macOS 上使用 `Cmd+\`，在 Windows 和 Linux 上使用 `Ctrl+\` 切换原始模式。

## 目录

目录面板根据当前笔记中的标题生成大纲。对于长笔记、操作流程、研究文件和生成的文档非常有用。在 macOS 上使用 `Cmd+Shift+T`，在 Windows 和 Linux 上使用 `Ctrl+Shift+T` 切换显示。

## 宽度

笔记可以使用普通或宽屏编辑器宽度。在设置中设置默认值，或在编辑器工具栏中为单个笔记覆盖此设置。
