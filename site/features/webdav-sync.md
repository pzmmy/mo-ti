# WebDAV 同步

墨屉支持通过 WebDAV 协议将笔记同步到远程服务器，方便你在不同设备之间存取笔记内容。

## 配置步骤

1. 打开 **设置** → **WebDAV 同步** 部分
2. 开启 **启用 WebDAV 同步** 开关
3. 填写以下配置信息：

| 配置项 | 说明 | 示例 |
|--------|------|------|
| **服务器地址** | WebDAV 服务器的完整 URL | `https://example.com/dav/` |
| **用户名** | WebDAV 认证用户名 | `username` |
| **密码** | WebDAV 认证密码 | — |
| **远程路径** | 服务器上存放笔记的目录路径 | `/mo-ti-vault`（默认值） |

4. 点击 **测试连接** 验证配置是否正确
5. 点击 **立即同步** 开始同步

### 密码安全

密码会使用系统密钥加密存储在 `webdav-passwords.enc` 文件中，不会以明文形式持久化。

## 同步进度条

同步过程中会显示实时的进度信息，包含以下阶段：

| 阶段 | 说明 |
|------|------|
| 连接中 | 正在连接 WebDAV 服务器 |
| 确保远程目录 | 检查并创建远程目录结构 |
| 收集文件 | 扫描本地和远程笔记文件 |
| 上传中 | 将本地新增/修改的笔记上传到服务器 |
| 下载中 | 将远程新增/修改的笔记下载到本地 |
| 完成 | 同步完成 |

进度条会显示当前处理的文件序号和百分比。在连接和收集阶段，进度条以无限动画形式显示。

## 移动伴侣

墨屉提供了一个移动端 Web 界面（`/mobile/`），可以通过 WebDAV 在手机上访问你的笔记。

### 在手机上使用

1. 确保你的 WebDAV 服务器可以从外部网络访问
2. 在手机浏览器中打开 `https://你的域名/mobile/`
3. 输入相同的 WebDAV 服务器地址、用户名和密码
4. 连接成功后即可浏览、阅读和编辑笔记

移动端支持：
- 📖 浏览笔记列表，支持搜索过滤
- ✏️ 编辑 Markdown 笔记
- 💾 保存修改回 WebDAV 服务器
- 🌙 跟随系统深色模式

记者的凭据仅保存在浏览器会话中（sessionStorage），关闭页面后自动清除。
