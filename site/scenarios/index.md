# 场景指南

墨屉的应用场景远不止笔记记录。从备考复习到职场知识管理，从知识付费课程整理到家庭小团队协作，墨屉的本地优先、Markdown 原生和 Git 版本管理能力，能帮你搭建真正属于你的知识体系。

选择以下场景，快速上手：

- [考研/考公备考笔记](/scenarios/exam-prep) — 用墨屉构建高效的备考知识库
- [职场个人知识库](/scenarios/daily-work) — 搭建你的第二大脑，提升工作效率
- [知识付费课程管理](/scenarios/knowledge-pay) — 系统化整理你购买的课程笔记
- [家庭/小团队知识库](/scenarios/team-knowledge) — 多人共享知识的协作方案

每个场景包含：
1. **场景描述** — 适用人群与痛点
2. **操作指南** — 5–8 步具体步骤
3. **功能清单** — 使用的墨屉功能及对应文档
4. **模板推荐** — 可复用的 Markdown 模板

> 提示：这些场景均基于本地文件，你的数据完全由你掌控，无需担心平台锁死或隐私泄露。
